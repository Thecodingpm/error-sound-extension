# Sounds Folder

Drop your `.mp3` sound files here. The extension expects these filenames:

| File | Sound |
|---|---|
| `vineboom.mp3` | Vine Boom |
| `metalpipe.mp3` | Metal Pipe |
| `faaah.mp3` | Faaah |
| `xp-error.mp3` | Windows XP Error |

You can source royalty-free versions from:
- https://freesound.org
- https://www.myinstants.com
- https://soundbible.com

After adding files, reload the Extension Development Host (Ctrl+R).
